# Meme Generator

Aplicación web interactiva para crear memes personalizados.  
Permite agregar texto superior e inferior a una imagen y aplicar filtros visuales en tiempo real.

🔗 Live Demo  
https://melina8444.github.io/editor_meme/

---

## 🚀 Características

- Edición de texto superior e inferior
- Aplicación de filtros en imágenes
- Manipulación dinámica del DOM
- Vista previa en tiempo real
- Interfaz simple y responsive

---

## 🛠 Tecnologías utilizadas

- JavaScript
- HTML5
- CSS3
- Manipulación del DOM
